# tinyWeather
A quick little weather app.

<h2>How to use:</h2>
<ul>
  <li>`Clone repo`</li>
  <li>Install packages: npm i</li>
  <li>Start up: npm start</li>
</ul>

<h2>Run the build</h2>
<ul>
  <li>npm run build</li>
</ul>

<h2>Test the app</h2>
<p>tinyWeather can be tested with jest</p>
<ul>
  <li>npm test</li>
</ul>
