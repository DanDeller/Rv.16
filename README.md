# tinyWeather
A little weather app.

<h2>How to use:</h2>
<ul>
  <li>Clone the repo</li>
  <li>Install packages: yarn</li>
  <li>Start the server: yarn server</li>
  <li>Start the app: yarn start</li>
</ul>

> NOTE: If you are not using yarn, you can still use npm in its place.
